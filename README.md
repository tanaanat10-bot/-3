Цей проєкт демонструє роботу з базою даних SQLite у Python. У ньому створюється база з чотирма таблицями: Users, Orders, Products та OrderItems. Далі таблиці наповнюються тестовими даними за допомогою INSERT. Після цього виконуються два SELECT-запити з використанням JOIN, WHERE, GROUP BY та HAVING для отримання аналітичної інформації про користувачів, замовлення та товари. Результати запитів виводяться у консоль.

Також у проєкті виконується оновлення даних за допомогою UPDATE, після чого SELECT-запити повторно запускаються, щоб показати, як змінився результат у базі даних. Увесь функціонал зібрано в одному Python-файлі, який послідовно створює базу, додає дані, виконує запити та виводить результати.

Цей проєкт підходить для ознайомлення з основами роботи з реляційними базами даних, виконанням SQL-запитів у Python та демонструє базові операції CRUD (Create, Read, Update).
